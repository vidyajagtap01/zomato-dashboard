# zomato-dashboard
This project presents an interactive Zomato Analytics Dashboard built using Power BI. It provides deep insights into sales performance, customer behavior, city-level trends, and user demographics.
The dashboard is divided into three major sections:

📊 Sales Dashboard

👥 User Analysis

🌍 Area Analysis

🎯 Business Objective

To analyze Zomato’s:

Sales performance

Customer acquisition & churn

City-wise revenue trends

Ratings and order distribution

Demographic user insights

And generate actionable insights for business decision-making.

🛠️ Tools & Technologies

Power BI Desktop

DAX (Data Analysis Expressions)

Data Modeling

Excel / CSV Dataset

Data Cleaning & Transformation (Power Query)

📊 Dashboard Features
1️⃣ Sales Dashboard

Total Sales Amount: 987M

Total Quantity: 2M

Total Orders: 150K

Total Ratings: 148K

Yearly Sales Trend Analysis

Top 10 Cities by Sales

Category-wise Sales (Veg / Non-Veg / Other)

Dynamic slicers (Top 5, Top 10, Top 20, etc.)

2️⃣ User Analysis Dashboard

Active Users: 78K

Total Users: 78K

Orders: 301K

Gain vs Lost Customers

Gender-based customer comparison

Age-wise user distribution

City search functionality

3️⃣ Area Analysis Dashboard

Sale value by city

Rating count by city

Year-wise top sales distribution

Quantity vs Amount toggle

Detailed city-level table view

📈 Key Insights

📍 Electronic City generated the highest sales.

📈 Sales peaked in 2018.

👨 Male customers showed slightly higher acquisition and churn compared to female customers.

🍔 Veg category contributed more revenue than Other category.

👥 Majority users fall within the 23–26 age group.
